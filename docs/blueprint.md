# **App Name**: AccountSelectr

## Core Features:

- Account Type Display: Render different account types (Savings, Custody, Numbered, Cryptocurrency) with their respective descriptions, compatibility tags (e.g., SWIFT, ETF), and icons.
- Single Account Selection: Allow users to select only one account type from the available options using a radio button-like interface.
- Informational Banner: Display a persistent informational banner regarding ETT transaction approvals and a glossary reference at the bottom of the form.
- Form Navigation Controls: Provide 'Save & Continue Later' and 'Next: Personal Details' buttons for navigating the multi-step form.

## Style Guidelines:

- Primary color: A deep, professional navy blue (#142952), conveying trust and stability.
- Background color: A very light, subtle grey-blue (#F0F2F4), providing a clean and understated canvas.
- Accent color: A vibrant aqua blue (#3CC2DD), offering a refreshing contrast and drawing attention to interactive elements.
- Headline font: 'Alegreya' (serif) for prominent titles like 'Select Account Type', providing an elegant and intellectual feel.
- Body font: 'PT Sans' (sans-serif) for all other text, ensuring clear readability with a modern, warm touch.
- Use distinct, modern, and stylized icons for each account type (e.g., green building for Savings, red shield for Custody, golden padlock for Numbered, grey Bitcoin symbol for Crypto), and a clear 'info' icon for the advisory banner.
- Utilize a responsive two-column grid layout for the account selection cards, clearly segmenting the header, selection options, information banner, and navigation footer. Each account option will be presented within a distinct card.
- Implement subtle hover effects and a clear visual highlight (e.g., border change, subtle background shift) upon selection of an account type card, providing immediate user feedback.
